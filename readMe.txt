Setps to run node frontend and backend
------------------------------------------
1.Two folder exits name backend and frontend to go terminal and install npm packages
	inside both folder by running this command "npm install"
2.Once done open path for backend and frontend in terminal and run "npm start"
3.you will get port for both frontend and backend
4.run website through port as "http://localhost:3001" and "http://localhost:3232"
